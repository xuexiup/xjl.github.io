安装pycharm软件后在Terminal终端无法使用pip，显示No module named pip如何解决?

安装pycharm的时候会卸载pip的有关内容。若出现了这种情况，则运行以下指令：

1）python -m ensurepip
2）python -m pip install –upgrade pip
