Js给数组对象（每个）添加一个新元素/字段并赋值

1.数组.map

map() 方法返回一个新数组，数组中的元素为原始数组元素调用函数处理后的值。

map() 方法按照原始数组元素顺序依次处理元素。

注意： map() 不会对空数组进行检测。

注意： map() 不会改变原始数组。

用法：array.map(function(currentValue,index,arr), thisValue)

currentValue 必须。当前元素的值
index 可选。当前元素的索引值
arr 可选。当前元素属于的数组对象

1.array.push()

在数组尾部添加新的元素，并返回新的数组长度。
