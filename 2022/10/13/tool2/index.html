VScode常用的插件集合

  1. Vetur/Volar ：vue/vue3语法高亮、智能感知、Emmet等, 包含格式化功能， Alt+Shift+F （格式化全文）
  2. ESLint : 校验js代码规范
  3. Prettier - Code formatter ： 格式化js、css代码插件
  4. open in browser ： 在文件右键 可以选择在浏览器预览
  5. Auto Rename Tag : 重命名一个HTML / XML标签时，自动重命名配对的HTML / XML标签
  6. Path Intellisense ：路径自动补全插件
  7. Bracket Pair Colorizer ：各种结对的括号颜色高亮
  8. HTML CSS Support ： html 标签上 class 智能提示当前项目所支持的样式
  9. HTML Snippets： H5代码片段以及提示
  10. CSS Peek： css 样式追踪
  11. filesize： 在底部状态栏显示当前文件大小，点击后还可以看到详细创建、修改时间
  12. REST Client ：优雅调试接口
  13. npm Intellisense：导入语句自动补全npm模块名称
  14. Code Runner ：代码一键运行，支持超过40种语言
  15. Color Info ： CSS 中使用颜色得色彩模型的（HEX、 RGB、HSL 和 CMYK）
  16. Format Files ： 一键格式化目录下所有代码
  17. image preview ：鼠标hover图片路径上，显示图像预览
  18. JavaScript Booster ：代码重构，简化插件（把var转为const或者let，去除多余的else语句，合并声明和初始化）
  19. Debugger for Chrome：编辑器中打断点，轻松地在Chrome里调试JavaScrip
  20. GitLens ：丰富的git日志插件
  21. Material Icon Theme：Material主题图标
  22. Sea Green Theme：主题
  23. Chinese (Simplified) Language Pack for Visual Studio Code：中文（简体）语言包为VS Code提供本地化界面
  24. Alignment：根据=、:、*=、+=、=>、/=等符号对选中代码快速对齐
  25. Debug Visualizer：实现算法可视化插件
  26. Settings Sync：同步 Visual Studio Code 设置
  27. Turbo Console Log：快捷console.log
  28. LeetCode：刷题网站 LeetCode插件
  29. :emojisense: ：快速挑选 Markdown 中的 Emoji。
  30. Live Server：加载功能的本地开发服务器，以处理静态和动态页面。
